# CSM-ModSets-SplashWindow

基于 [CSM（可通信状态机）](https://nevstop-lab.github.io/CSM-Wiki/) 框架的启动引导窗口模块，用于在系统启动过程中显示友好的加载过渡界面。

This repository contains a CSM-based splash screen module for displaying a friendly loading transition during system startup.

---

## 模块列表

| 模块 | 所属模块组 | 功能说明 |
| --- | --- | --- |
| [SplashWindow](./SplashWindow.md) | CSM-SplashWindow.lvlib | 启动引导窗口，显示模块启动进度 |
